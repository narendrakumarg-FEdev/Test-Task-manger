Welcome to the Task Manager App!

This application is designed to help you manage your tasks efficiently. Below are some key features you can take advantage of the apllication:

Features

Add Tasks

Easily create new tasks by clicking the "Add Task" button. Fill in the task details and save it to your list.

Manage Tasks

View your tasks in a list format. You can edit or delete tasks as needed.

Change Status

Update the status of your tasks to reflect their current state (e.g., Pending, In Progress, Completed).

Filter Tasks

Use filters to view tasks based on their status or due dates.

Notifications

Receive reminders for tasks that are due soon.
